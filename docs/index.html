Timeline

The story of SpecBot

24 December 2018

The idea was born

I (The76i) was moderating and improving System32Comics' Discord server when I had a bot idea. I knew if I needed a Discord bot, I should contact oko123, and I did.

25 December 2018

The skeleton of the bot was done

The following day, 25.12.2018 oko had already completed the skeleton of the bot and we created a test server for it. We started testing internally before inviting it to any other servers. It was awesome, something you had planned, actually worked.

26 December 2018

SpecBot 1.0 was ready and privately used for the first time

SpecBot 1.0 was ready without major bugs, so we decided to invite it to System32Comics' Discord server,  it had a few bugs but they were fixed quickly. It was used by many people at the same time and worked perfectly.

27 December 2018

We started planning to make the bot publicly available

The bot was so fun to play with, so I suggested we should make the bot public. We started planning about new features and  oko re-coded the whole thing, now with support for lots of more things.

28 December 2018

Bot was getting more and more ready for release

The bot was basically ready to be released. I set up this site while oko was coding even more new stuff and making the bot even better. We created the installation guide and did testing on Windows and Ubuntu.

Why SpecBot?

Isn't it annoying when every time you post a picture of your PC on some discord server, someone will ask always the same question: "Specs?". That's why we created SpecBot. You can just say: "-PC1 @user#0000" and SpecBot will list your specifications! Now you have to type out your specs only once!
